상세 설명 (Inspector)

흐름도의 각 단계를 클릭하면 이곳에 상세 설명이 나타납니다.

index.jsp

메인 페이지입니다. `header.jsp`를 include하며, 로그인 상태에 따라 다른 메뉴를 보여줍니다.

register.jsp

회원가입 폼(View)입니다. 사용자가 ID, PW, 이름을 입력합니다.

register.do (RegisterServlet)

Controller입니다. 폼 데이터를 받아 `UserDAO`를 통해 DB에 `INSERT`합니다.

완료 후 `login.jsp`로 `Redirect` (PRG 패턴)합니다.

login.jsp

로그인 폼(View)입니다.

login.do (LoginServlet)

Controller입니다. `UserDAO`로 `SELECT` 검증을 수행합니다.

성공 시: `HttpSession`을 생성하고 'userId'를 저장한 뒤 `index.jsp`로 `Redirect`합니다.

실패 시: `login.jsp`로 `Redirect`합니다.

index.jsp (로그인 상태)

`sessionScope.userId`가 존재하므로 `header.jsp`가 '내 장바구니', '상품 등록' 메뉴를 활성화합니다.

product_list.do (ProductListServlet)

Controller입니다. `ProductDAO`를 통해 DB에서 `SELECT *` (상품 목록)을 가져옵니다.

가져온 `List`를 `request.setAttribute`에 담아 `product_list.jsp`로 `Forward`합니다.

product_list.jsp

상품 목록(View)입니다. JSTL ``를 사용해 Controller가 넘겨준 `requestScope.productList`를 화면에 렌더링합니다.

product_upload.jsp

상품 등록 폼(View)입니다. `enctype="multipart/form-data"` 속성이 필수입니다.

product_upload.do (ProductUploadServlet)

Controller입니다. `commons-fileupload` 라이브러리를 사용해 폼 데이터(텍스트)와 '파일'을 분리하여 파싱합니다.

파일은 `C:/jsp_upload` 경로에 `UUID`로 저장하고, 파일 정보와 폼 데이터는 `ProductDAO`를 통해 DB에 `INSERT`합니다.

완료 후 `product_list.do`로 `Redirect`합니다.

cart.do (CartServlet)

Controller입니다. 세션에서 `userId`를, 파라미터에서 `prodId`를 가져옵니다.

`CartDAO`를 통해 `Cart` 테이블에 `INSERT` (장바구니 담기)를 수행합니다.

완료 후 `cart_list.do`로 `Redirect`합니다.

cart_list.do (CartListServlet)

Controller입니다. 세션에서 `userId`를 가져옵니다.

`CartDAO`를 통해 `Cart`와 `Product` 테이블을 `JOIN`하여 '내 장바구니 목록'을 `SELECT`합니다.

결과 `List`를 `request.setAttribute`에 담아 `cart_list.jsp`로 `Forward`합니다.

order.do (OrderServlet)

Controller입니다. (모의 결제)

`CartDAO`를 통해 `DELETE FROM Cart WHERE user_id = ?` (장바구니 비우기)를 실행합니다.

완료 후 `order_complete.jsp`로 `Redirect`합니다.

order_complete.jsp

결제 완료(View)입니다. 사용자에게 완료 메시지를 보여줍니다.

logout.do (LogoutServlet)

Controller입니다. `session.invalidate()`를 호출하여 세션을 파기합니다.

완료 후 `index.jsp`로 `Redirect`합니다.
